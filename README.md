# Portal Fatigue - KubeCon EU 2026

Abhinav's KubeCon EU 2026 talk on portal fatigue.

## Overview

This repository contains materials for the KubeCon EU 2026 presentation on portal fatigue — the cognitive and operational overhead developers face when navigating too many internal developer portals, dashboards, and self-service platforms.
